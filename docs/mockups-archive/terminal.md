# Terminal (Kitty)

## Base Colors

- Background: `#171b1a`
- Foreground: `#ccc8c2`
- Cursor: `#ccc8c2` (white beam, not orange)
- Selection: intellect blue `#5bc0d6` bg with dark text `#171b1a`

## Starship Prompt

```
~/my-rice  main !3 +1
>
```

| Element | Color | Font |
|---------|-------|------|
| Directory | `#5bc0d6` bold (intellect blue) | — |
| Git branch | `#7555c6` bold (psyche purple) | — |
| Git status (dirty) | `#eb6408` bold (action orange) | — |
| `>` character (success) | `#4b4b4b` (disabled gray) | — |
| `>` character (error) | `#b83a3a` (error red) | — |

## 16-Color Palette

| Index | Name | Normal | Bright |
|-------|------|--------|--------|
| 0/8 | Black | `#171b1a` | `#363b3a` |
| 1/9 | Red | `#cb456a` (physique) | `#eb6408` (action orange) |
| 2/10 | Green | `#0fb666` (success) | `#3dcc88` |
| 3/11 | Yellow | `#e3ba3e` (motorics) | `#eece6e` |
| 4/12 | Blue | `#5bc0d6` (intellect) | `#89d4e5` |
| 5/13 | Magenta | `#7555c6` (psyche) | `#9a7ed6` |
| 6/14 | Cyan | `#5bc0d6` (intellect) | `#89d4e5` |
| 7/15 | White | `#ccc8c2` | `#ffffff` |

Blue and cyan share intellect blue. Terminal red = physique, magenta = psyche, yellow = motorics.

## Font

JetBrainsMono Nerd Font, 12pt, 8px padding.
